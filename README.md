# Django Multivendor Platform

This is a **Django-based Multivendor E-commerce Platform** that allows multiple vendors to register, add products, and manage their sales within a single platform. The project is built using **Django**, **Python**, and **PostgreSQL** for database management. 

Features include user registration, vendor management, product management, shopping cart functionality, and order management. The platform supports multiple vendors with separate dashboards for admins and vendors. The backend handles user authentication, CRUD operations for products, and secure management of orders and payments.

I developed this project from scratch, including setting up the Django project structure, creating models, views, templates, forms, and database migrations. The project also includes writing and running test cases to ensure code reliability. This repository demonstrates hands-on experience in **Django development**, database integration, and implementing real-world e-commerce functionality.
